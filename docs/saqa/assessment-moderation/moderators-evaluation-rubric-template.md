## Scoring Scale

Apply the following scale to every criterion in the rubric below.

| Score | Label | Description |
|:-----:|-------|-------------|
| **4** | Fully Meets Requirements | The criterion is fully and consistently met. No corrective action is required. |
| **3** | Substantially Meets Requirements | The criterion is mostly met with minor deficiencies that do not significantly affect the validity, reliability, or fairness of the assessment. A minor improvement recommendation may be noted. |
| **2** | Partially Meets Requirements | The criterion is partially met. Significant deficiencies are present that affect the quality of the assessment. Corrective action is required. |
| **1** | Does Not Meet Requirements | The criterion is not met or is seriously deficient. Immediate corrective action is required. |
| **N/A** | Not Applicable | The criterion does not apply to this assessment context. Provide a justification in the notes column. |

**Minimum Acceptable Score:** A criterion that receives a score of **1** requires a mandatory corrective action and must be re-moderated after correction.

**Overall Outcome Guidance:**
- **Average score ≥ 3.5 across all applicable criteria:** Moderation outcome — **Acceptable (A)**
- **Average score ≥ 2.5 and < 3.5:** Moderation outcome — **Acceptable with Recommendations (AWR)**
- **Average score < 2.5 or any criterion scored 1:** Moderation outcome — **Not Acceptable — Corrective Action Required (NACAR)**

## Rubric Section 1 — Assessment Instrument Quality

These criteria evaluate the quality of the assessment instruments (knowledge tests, observation checklists, portfolio guides, etc.) used by the assessor.

| # | Criterion | Score (1–4 / N/A) | Evidence / Observations | Corrective Action Required (Y/N) |
|---|-----------|:-----------------:|------------------------|:--------------------------------:|
| 1.1 | **Validity:** The assessment instruments clearly and directly assess the specific outcomes and assessment criteria of the registered unit standards. | | | |
| 1.2 | **Fairness:** The instruments are free from cultural, linguistic, or other bias. Language and format are accessible to the target learner group. | | | |
| 1.3 | **Reliability:** The instruments are sufficiently standardised that different assessors applying them to the same learner evidence would reach the same conclusion. Model answers or scoring rubrics are provided where appropriate. | | | |
| 1.4 | **Sufficiency:** The instruments together provide adequate opportunity for the learner to demonstrate competence across all outcomes and assessment criteria. | | | |
| 1.5 | **Alignment to Outcomes / Unit Standards:** Each assessment instrument clearly references the unit standard(s) and specific outcome(s) it assesses. | | | |
| 1.6 | **Currency:** The instruments reflect the current version of the unit standards and qualification requirements (not outdated or superseded). | | | |
| 1.7 | **Practicability:** The instruments are logistically feasible to administer within the stated assessment context. | | | |

## Rubric Section 2 — Assessor Judgment and Decision-Making

These criteria evaluate the consistency and correctness of the assessor's judgments.

| # | Criterion | Score (1–4 / N/A) | Evidence / Observations | Corrective Action Required (Y/N) |
|---|-----------|:-----------------:|------------------------|:--------------------------------:|
| 2.1 | **Consistency:** The assessor's judgments are consistent across similar learner responses and evidence items. Where multiple portfolios are reviewed, decisions are not arbitrary or contradictory. | | | |
| 2.2 | **Correct Application of Decision Rules:** The assessor has correctly applied the Competent / Not Yet Competent decision rules as specified in the Assessor's Guide. | | | |
| 2.3 | **Evidence Mapping:** The assessor has clearly mapped learner evidence to specific outcomes and assessment criteria in the marking/scoring instruments. | | | |
| 2.4 | **Judgments Are Evidence-Based:** Assessment decisions are supported by documented evidence; speculative or undocumented conclusions are absent. | | | |
| 2.5 | **Handling of Borderline Cases:** Where learner performance is close to the pass threshold, the assessor has applied judgment consistently and documented their reasoning. | | | |

|---|-----------|:-----------------:|------------------------|:--------------------------------:|
