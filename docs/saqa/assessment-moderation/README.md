# SAQA/HWSETA Assessment & Moderation Templates

This folder contains standardised assessment and moderation documentation templates aligned with the South African Qualifications Authority (SAQA) and the Health and Welfare Sector Education and Training Authority (HWSETA) requirements.

---

## Templates in This Folder

| File | Purpose |
|------|---------|
| [`assessors-guide-template.md`](assessors-guide-template.md) | Assessor's Guide — marking and scoring framework for assessors |
| [`integrated-assessment-matrix-template.md`](integrated-assessment-matrix-template.md) | Integrated Assessment Matrix — maps rubric criteria to NQF descriptors and unit standards |
| [`moderators-evaluation-rubric-template.md`](moderators-evaluation-rubric-template.md) | Moderator's Evaluation Rubric — external moderation quality-assurance tool |
| [`external-moderation-report-template.md`](external-moderation-report-template.md) | External Moderation Report — standardised reporting form for external moderators |

---

## Program Tracks Supported

These templates support two qualification/program tracks:

| Track | Target Audience | NQF Level |
|-------|----------------|-----------|
| **Peer Counsellor Program** | Peer counsellors and support workers | NQF Level 4 |
| **Facilitator Program** | Facilitators and program trainers | NQF Level 5 |

Each template includes clearly labelled sections or columns for **Level 4** and **Level 5** usage. Where separate documents are more appropriate, the template heading instructs the maintainer to duplicate and adapt the file for each level.

> **Maintainer Note:** Before distributing templates, search for all placeholders (e.g., `UNIT_STANDARD_ID`, `QUALIFICATION_ID`, `INSTITUTION_NAME`) and replace them with the actual registered values from the SAQA National Learners' Records Database (NLRD).

## Assessment & Moderation Workflow

Use the following sequence for every cohort assessment cycle:

```
Pre-Assessment
    │
    ▼
Assessment
    │
    ▼
Internal Moderation
    │
    ▼
External Moderation
    │
    ▼
Corrective Actions & Re-assessment (if required)
    │
    ▼
Record-keeping & Certification
```

### Step 1 — Pre-Assessment

**Who:** Assessor (with oversight from the Assessment Coordinator)

**Actions:**
1. Confirm learner eligibility and readiness.
2. Brief learners on the assessment purpose, instruments, and process.
3. Complete the **Pre-Assessment Checklist** in the Assessor's Guide.
4. Ensure the assessment venue/context meets SAQA requirements (safe, accessible, free from bias).
5. Prepare all assessment instruments and evidence-collection tools.

**Documents used:** Assessor's Guide (Pre-Assessment Checks section)

---

### Step 2 — Assessment

**Who:** Registered Assessor

**Actions:**
1. Administer each assessment instrument (knowledge, practical, workplace/portfolio).
2. Collect and record evidence against rubric criteria.
3. Complete scoring tables and apply decision rules (Competent / Not Yet Competent).
4. Provide written feedback to the learner.

**Documents used:** Assessor's Guide, Integrated Assessment Matrix

---

### Step 3 — Internal Moderation

**Who:** Internal Moderator (qualified moderator employed by the institution)

**Actions:**
1. Sample a minimum of 10 % (or as required by HWSETA) of assessed portfolios.
2. Evaluate assessment instruments for validity, fairness, and reliability.
3. Check that assessor judgments are consistent and evidence is sufficient.
4. Document findings and return to the assessor for corrections, if required.

**Documents used:** Moderator's Evaluation Rubric, External Moderation Report (adapted for internal use)

---

### Step 4 — External Moderation

**Who:** HWSETA-appointed External Moderator

**Actions:**
1. Receive the full assessment pack (instruments, evidence, assessor records).
2. Select a representative sample using the rationale documented in the Moderation Report.
3. Complete the Moderator's Evaluation Rubric for each sampled portfolio.
4. Compile the External Moderation Report including non-conformances and recommendations.
5. Sign off and submit to HWSETA and the institution.

**Documents used:** Moderator's Evaluation Rubric, External Moderation Report

---

### Step 5 — Corrective Actions & Re-assessment

**Who:** Assessor, Internal Moderator, Assessment Coordinator

**Actions:**
1. Review non-conformances and recommendations from the External Moderation Report.
2. Implement corrective actions within the timeframe specified by HWSETA.
3. Arrange re-assessment for learners who received a "Not Yet Competent" outcome, where applicable.
4. Document corrective actions and retain evidence for audit purposes.

---

### Step 6 — Record-keeping & Certification

**Who:** Assessment Coordinator / Quality Assurance Manager

**Actions:**
1. Capture learner results in the institution's learner management system.
2. Submit achievement data to HWSETA for upload to the SAQA NLRD.
3. Issue certificates or statements of results.
4. Archive all assessment and moderation documentation for the period required by HWSETA (minimum five years).

## Adapting Templates for Level 4 vs Level 5

- **NQF Level 4 (Peer Counsellor):** Use the Level 4 sections/columns. The NQF Level 4 descriptor emphasises *application of knowledge in a defined context with some degree of responsibility for others*.
- **NQF Level 5 (Facilitator):** Use the Level 5 sections/columns. The NQF Level 5 descriptor emphasises *application of knowledge in a specialised context, with significant responsibility for the work of others*.

When completing the Integrated Assessment Matrix, map each rubric criterion to the appropriate NQF level descriptor and insert the correct `UNIT_STANDARD_ID` values from the SAQA NLRD.
