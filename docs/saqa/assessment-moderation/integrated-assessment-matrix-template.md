## NQF Level Descriptors Reference

Use these descriptors when completing the **NQF Level Descriptor Mapping** column.

### NQF Level 4 — Peer Counsellor Program

> Source: SAQA NQF Policy and Criteria (latest edition). Replace with the exact current descriptor if it has been updated.

| Descriptor Dimension | NQF Level 4 Description |
|---------------------|------------------------|
| **Knowledge** | Learners demonstrate knowledge of and engagement with facts, principles, and processes in a familiar, defined context. Knowledge is concrete and may include some abstract ideas and concepts. |
| **Skills** | Learners apply a range of procedures and processes, selecting the most appropriate and explaining their selection. |
| **Applied Competence** | Learners integrate knowledge and skills in a range of familiar and some unfamiliar contexts, demonstrating some originality. |
| **Autonomy & Responsibility** | Learners act with some degree of responsibility for their own learning and that of others. |

### NQF Level 5 — Facilitator Program

| Descriptor Dimension | NQF Level 5 Description |
|---------------------|------------------------|
| **Knowledge** | Learners demonstrate knowledge of and engagement with a body of knowledge that is appropriate to a field of study or work. Knowledge includes abstract theoretical components as well as practical application. |
| **Skills** | Learners apply a range of complex procedures and processes in both familiar and unfamiliar contexts, justifying their approach. |
| **Applied Competence** | Learners integrate knowledge and skills to formulate responses to defined problems in complex and unfamiliar contexts. |
| **Autonomy & Responsibility** | Learners act with significant personal autonomy and responsibility for the work of others. |

## Column Definitions

| Column | Description |
|--------|-------------|
| **Module / Unit** | The training module or unit within the qualification. |
| **Assessment Activity** | The specific assessment task or instrument (e.g., knowledge test, observation, portfolio). |
| **Rubric Criterion** | The specific performance criterion being assessed. |
| **Evidence Required** | The type and description of evidence the learner must produce. |
| **NQF Level** | 4 or 5 (or Both). |
| **NQF Level Descriptor Mapping** | Which NQF descriptor dimension(s) the criterion aligns to (Knowledge / Skills / Applied Competence / Autonomy & Responsibility). |
| **Unit Standard / Outcome Mapping** | The SAQA unit standard ID(s) and specific outcome(s) the criterion assesses. |
| **Weighting (%)** | The percentage weighting of this criterion within the assessment activity. |
| **Assessment Method** | How the criterion is assessed (e.g., observation, written response, product review, oral questioning). |
| **Instrument Reference** | Reference to the specific assessment instrument document (e.g., Appendix B, Observation Checklist v1.0). |
| **Moderation Notes** | Guidance for moderators reviewing this criterion (e.g., common assessor errors, clarifications, sampling priority). |

## Notes for Assessors and Moderators

- **Weighting:** Ensure that the total weighting for each assessment activity sums to 100%.
- **Unit Standard Mapping:** Each rubric criterion should be mapped to at least one unit standard outcome. Where a criterion spans multiple outcomes, list all relevant references separated by a semicolon.
- **NQF Descriptor Mapping:** A single criterion may align to multiple NQF descriptor dimensions. List all applicable dimensions.
- **Instrument Reference:** Always include the document title, version, and date in the instrument reference column to ensure traceability.
- **Moderation Notes:** Use this column to flag criteria that have historically been misapplied or require particular attention during sampling.
