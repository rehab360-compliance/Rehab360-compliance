# Outpatient Treatment Curriculum – Universal Treatment Curriculum (UTC)

This folder contains the complete UTC-based curriculum for Rehab360's outpatient Substance Use Disorder (SUD) treatment programme. The curriculum is structured around the ten-module Universal Treatment Curriculum framework, adapted for the South African context and aligned to HWSETA unit standards.

## Curriculum Structure

The UTC is designed to be delivered sequentially, with later modules building on concepts introduced in earlier ones. Modules 1–6 are required for NQF Level 4 (Peer Counseling) and Modules 1–10 are required for NQF Level 5 (Substance Abuse Counselling Practitioner).

## Delivery Mode

All modules are designed for delivery in an outpatient group treatment setting, supplemented by individual counselling sessions and practical placement hours.
