# Assessment – Outpatient Treatment Curriculum

This folder contains assessment guide templates and Portfolio of Evidence (PoE) checklists for Rehab360's UTC-based outpatient SUD treatment programme.

## Documents

| File | Description |
|------|-------------|
| [assessment-guide-template.md](assessment-guide-template.md) | Standardised assessment guide template for each UTC module — includes formative tasks, summative tasks, marking criteria, and moderation instructions |
| [poe-checklist.md](poe-checklist.md) | Learner Portfolio of Evidence (PoE) compilation checklist — ensures all required evidence is present before internal moderation |

## Usage

1. Complete the [Assessment Guide Template](assessment-guide-template.md) for each UTC module before the intake cycle begins.
2. Issue the PoE checklist to learners at the start of the programme.
3. Refer to the [Accreditation Evidence Alignment Table](../saqa/assessment-moderation/accreditation-evidence-alignment-table.md) to confirm all HWSETA evidence criteria are met.
4. Complete internal moderation using the Moderation Report (see [SAQA Assessment & Moderation](../saqa/assessment-moderation/README.md)) before submitting results to HWSETA.

## Assessor Requirements

- Registered assessor with HWSETA (NQF Level 5 or higher)
- Independent of facilitator role for summative assessment
- Familiar with all ten UTC modules and the relevant unit standards
