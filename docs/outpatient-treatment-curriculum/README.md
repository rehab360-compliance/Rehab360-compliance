## About the Universal Treatment Curriculum

The **Universal Treatment Curriculum (UTC)** is an evidence-based, standardised training framework developed to build competency in outpatient SUD treatment. It covers the clinical and relational skills required by practitioners at all levels — from peer counsellors through to registered substance abuse counsellors. Each UTC module maps directly to HWSETA unit standard outcomes and South African treatment guidelines.

## Usage

1. Orient new practitioners using the [UTC Module Overview](curriculum/utc-module-overview.md).
2. Deliver each module according to the relevant [Facilitator Session Plan](facilitator-resources/session-plan-template.md).
3. Issue learners the [Learner Guide Template](learner-resources/learner-guide-template.md) at the start of each module.
4. Collect formative and summative evidence per module using the [Assessment Guide Template](assessment/assessment-guide-template.md).
5. Compile completed evidence in the [PoE Checklist](assessment/poe-checklist.md) before internal moderation.
6. Complete moderation and submit results to HWSETA within the current reporting cycle (refer to the [Accreditation Evidence Alignment Table](saqa/assessment-moderation/accreditation-evidence-alignment-table.md)).
