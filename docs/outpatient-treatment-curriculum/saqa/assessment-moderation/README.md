# SAQA/HWSETA Assessment & Moderation – Outpatient Treatment Curriculum

This folder contains assessment, moderation, and accreditation evidence documents aligned to SAQA and HWSETA requirements for Rehab360's outpatient SUD treatment programme, delivered through the Universal Treatment Curriculum (UTC).

## Documents

| File | Description |
|------|-------------|
| [accreditation-evidence-alignment-table.md](accreditation-evidence-alignment-table.md) | Accreditation Evidence Alignment Table – maps required evidence to HWSETA criteria for NQF Level 4 (Peer Counseling) and NQF Level 5 (Substance Abuse Counselling Practitioner) in the outpatient treatment context |

## Qualification Levels Covered

- **NQF Level 4 – Peer Counseling in Substance Abuse (HWSETA)**
- **NQF Level 5 – Substance Abuse Counselling Practitioner (HWSETA)**

## UTC Alignment

Each accreditation criterion in this folder maps to one or more UTC modules. This ensures that evidence collected during programme delivery is directly traceable to both HWSETA unit standards and UTC competency benchmarks.

## Usage

1. Refer to the [Accreditation Evidence Alignment Table](accreditation-evidence-alignment-table.md) to identify which evidence items and templates are required for each HWSETA accreditation criterion.
2. Complete each linked template before the relevant intake cycle.
3. Store finalised portfolios in the organisation's document management system.
4. Submit completed results and reports to HWSETA within the current reporting cycle window.
