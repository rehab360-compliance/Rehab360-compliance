# Facilitator Resources – Outpatient Treatment Curriculum

This folder contains templates and guides for facilitators delivering the Universal Treatment Curriculum (UTC) in Rehab360's outpatient SUD treatment programme.

## Documents

| File | Description |
|------|-------------|
| [session-plan-template.md](session-plan-template.md) | Standardised session plan template for all UTC module sessions |
| [facilitator-guide-template.md](facilitator-guide-template.md) | Per-module facilitator guide template covering preparation, delivery notes, and debrief |

## Usage

1. Complete the [Session Plan Template](session-plan-template.md) for each session before delivery.
2. Use the [Facilitator Guide Template](facilitator-guide-template.md) to prepare module-specific guidance notes.
3. Retain all completed session plans in the learner group file as HWSETA evidence (see Criterion 3 in the [Accreditation Evidence Alignment Table](../saqa/assessment-moderation/accreditation-evidence-alignment-table.md)).
4. Submit session plans to the internal moderator for review before each intake cycle.

## Facilitator Requirements

- NQF Level 5 Facilitator registration with HWSETA (ETDP unit standard US 117871 or equivalent)
- Completed all ten UTC modules or equivalent clinical training
- Current registration with a relevant professional body (SACSSP, HPCSA, or HWSETA ETDP register)
