# Learner Resources – Outpatient Treatment Curriculum

This folder contains templates and guides for learners enrolled in Rehab360's UTC-based outpatient SUD treatment programme.

## Documents

| File | Description |
|------|-------------|
| [learner-guide-template.md](learner-guide-template.md) | Standardised learner guide template issued to learners at the start of each UTC module |

## Usage

1. Issue the completed [Learner Guide](learner-guide-template.md) to each learner at the start of every module.
2. Learners complete all exercises and reflections within the guide during and between sessions.
3. Completed learner guides form part of the formative evidence in the learner's Portfolio of Evidence (PoE) — see the [PoE Checklist](../assessment/poe-checklist.md).
4. Retain a copy of each issued and completed guide in the learner file.

## Learner Requirements

- Signed POPIA consent and learner registration form before first session
- Access to all ten UTC learner guides (one per module) throughout the programme
- Completed guides must be submitted to the assessor by the date specified in the relevant [Assessment Guide](../assessment/assessment-guide-template.md)
