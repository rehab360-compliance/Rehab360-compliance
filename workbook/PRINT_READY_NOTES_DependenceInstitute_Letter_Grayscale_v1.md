### Logo Concept (for designer implementation)

The logo conveys **hope, resilience, and African identity** while remaining professional and reproducible in monochrome.

**Visual elements:**
- **Central motif:** A stylised rising sun emerging from a traditional three-legged pot (potjie). The sun's rays transform into the outlines of a family/community holding hands, symbolising Ubuntu.
- **Colour palette (full colour version):**
  - Warm orange and gold — sunrise, energy, optimism
  - Deep green — growth, health, the land
  - Earth brown — grounding, tradition, the cooking pot
- **Grayscale variant:** Solid black icon + text; no gradients; all fills replaced with hatching or solid black.
- **Typography:** Clean modern sans-serif (Montserrat or Lato). "Dependence" slightly lighter weight; "Institute ZA" bold — softens the clinical feel.
- **Tagline placement:** "Rise. Heal. Thrive." in smaller text beneath the logo (can be omitted on tight layouts).

**Monogram alternative (for stamps/watermarks):** "DI ZA" in the same sans-serif, inside a simple circle.

## How to Export to PDF (Google Docs / Microsoft Word)

1. Copy the relevant workbook markdown file into Google Docs or Microsoft Word.
2. Replace every `---` (horizontal rule used as a page-break marker) with **Insert → Break → Page break** where a new page is intended.
3. **File → Page setup → Letter**; set all margins to **0.75"**.
4. Apply heading styles (Heading 1 / Heading 2 / Heading 3) matching the sizes above.
5. **File → Export → PDF** (Google Docs) or **File → Save As → PDF** (Word).
6. Print proof one copy in greyscale before distributing.
