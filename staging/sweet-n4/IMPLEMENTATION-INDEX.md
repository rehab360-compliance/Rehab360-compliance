## Usage Workflow

### Phase 1: Programme Setup (Before Learner Intake)
1. Complete `programme-design-document.md`
2. Customize `session-plan-template.md` for each session
3. Develop `learner-guide-template.md` for each module
4. Prepare `assessment-guide-template.md` for each unit standard
5. Submit for pre-assessment moderation

### Phase 2: Learner Enrollment
1. Issue `learner-registration-form.md` to each learner
2. Issue `learner-guide-template.md` for first module
3. Issue `poe-evidence-checklist.md`
4. Issue `portfolio-of-evidence-cover-sheet.md`
5. Communicate `appeals-and-grievances-policy.md`

### Phase 3: Programme Delivery
1. Facilitators use `session-plan-template.md`
2. Learners complete activities in `learner-guide-template.md`
3. Conduct formative assessments

### Phase 4: Workplace Learning
1. Place learners at approved sites
2. Issue `workplace-logbook-template.md`
3. Supervisors complete `supervisor-evaluation-form.md`

### Phase 5: Assessment
1. Assessors use `assessment-guide-template.md`
2. Complete `portfolio-of-evidence-cover-sheet.md`
3. Verify evidence using `poe-evidence-checklist.md`

### Phase 6: Moderation
1. Complete `internal-moderation-report-template.md`
2. Verify all assessment decisions
3. Approve for certification

### Phase 7: Reporting and Certification
1. Compile `results-schedule-template.md`
2. Submit to HWSETA
3. Issue certificates upon approval
