## Character & Naming Note

All programme materials use **Nkgono Mmatsela** / **Nkgono** for the elder/grandmother character who hosts *Nkgono Mmatsela's Kitchen* and leads the classroom pledge (Scene 9B). This is consistent across scripts, pause screens, pledge scenes, and workbook content.
